## üöÄ Quick Start

1. **Access the App**  
   [![Open in Browser](https://discordtextcolor.netlify.app/)

2. **Create Your Text**
   ```markdown
   1. Type or paste your text
   2. Highlight text sections
   3. Choose colors from the palette
   ```

3. **Copy to Discord**
   ```markdown
   ```ansi
   [1;31mRed Bold[0m and [44;37mBlue Background[0m text
   ```
   ```
